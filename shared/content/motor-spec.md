## 1. 概述

DM-4000 系列三相异步电动机是锐鹰传感最新一代高效节能工业电机产品。该系列电机采用先进的电磁设计和优化的冷却结构，符合 **IE4 超高效** 能效等级标准，广泛应用于风机、水泵、压缩机、输送机等工业驱动场景。

### 1.1 产品特点

- **超高效率**：满足 GB 18613-2020 / IEC 60034-30-1 IE4 能效等级
- **低噪声**：优化风道设计，运行噪声 ≤ 65 dB(A)
- **高可靠性**：F 级绝缘，B 级温升，设计寿命 ≥ 30,000 小时
- **灵活安装**：支持 B3、B5、B35 等多种安装方式
- **宽电压适应**：380V ± 10% 宽电压范围稳定运行

### 1.2 适用标准

| 标准编号 | 标准名称 |
|---------|---------|
| GB/T 755-2019 | 旋转电机 定额和性能 |
| GB 18613-2020 | 电动机能效限定值及能效等级 |
| IEC 60034-1 | 旋转电机 额定值和性能 |
| IEC 60034-30-1 | 旋转电机效率分级 |
| GB/T 12350-2022 | 小功率电动机的安全要求 |

### 4.1 性能说明

在额定工况下，DM-4000-7.5 电机的关键性能指标：

- **额定点效率**：93.2%，超过 IE4 标准要求的 92.1%
- **额定点功率因数**：0.87
- **最大效率点**：约 75% 负载时达到 93.8%
- **起动性能**：起动转矩为额定转矩的 2.5 倍，满足大多数工业负载直接起动需求

## 6. 使用环境与条件

| 条件 | 要求 |
|------|------|
| 海拔高度 | ≤ 1000m |
| 环境温度 | -15°C ~ +40°C |
| 相对湿度 | ≤ 95% (25°C) |
| 安装场所 | 无爆炸性气体、无腐蚀性气体 |
| 电源条件 | 三相 380V ± 10%, 50Hz ± 2% |

> **注意：** 在海拔超过 1000m 或环境温度超过 40°C 的条件下使用时，电机需要降额运行。具体降额系数请咨询技术支持。
